## Supported Platforms

Meshplay deploys as a set of Docker containers, which can be deployed to either a Docker host or Kubernetes cluster. See the complete list of supported platforms in the table below. With service meshes having sprung to life in the context of Kubernetes, so too, can Meshplayâ€™s deployment models be characterized in the context of Kubernetes. A given deployment of Meshplay can be described as either an _in-cluster_ or an _out-of-cluster_ deployment. Meshplay deploys as a stand-alone, management plane on a Docker host (_out-of-cluster_) or as a management plane in a Kubernetes cluster (_in-cluster_).
